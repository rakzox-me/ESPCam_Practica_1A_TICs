#Este proyecto consiste en crear una cámara inteligente pequeña y económica que puede "ver" y reconocer objetos por sí misma, sin necesitar estar conectada a internet.

Básicamente, le "enseñamos" a este dispositivo a diferenciar entre dos cosas: un encendedor y unas pinzas.

Funciona de la siguiente manera:

La cámara observa lo que tiene enfrente.

Si reconoce el encendedor o la pinza, nos avisa de tres formas al mismo tiempo:

Escribe el nombre del objeto en una pantallita.

Enciende una luz de color (verde o roja).

Hace un sonido (pitido).

Todo el sistema está protegido dentro de una caja impresa en 3D diseñada por nosotros para mantener la cámara firme.

En resumen, logramos que una pequeña placa electrónica actúe como un "ojo digital" que sabe distinguir qué herramientas le estamos mostrando.
